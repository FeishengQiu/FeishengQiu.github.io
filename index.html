SlidingMenu库的导入

1.github下载SlidingMenu的demo

https://github.com/jfeinstein10/SlidingMenu

2.解压进入该文件的libray目录，复制本地存放地址

E:\安卓参考的demo资料\北京新闻\1.上课资源\day1\资料\SlidingMenu-master\library

3.Android studio: New—>Import Moudle—>粘贴库的本地地址
—>改Module name(就是导入后库的名称) :SlidingMenu_library
—>点击Ok即可

4.修改SlidingMenu_library的build.gradle：
首先注释掉最上面的buidscript结构体，用不上；修改SDK和support库的版本即可，点击try again，出现一个bug：retrun (float)FloatMath.sin(f), 直接把FloatMath改成Math即可，修改完毕

5.关联库： 目录app右键—>Open Moudle Settings—>（右上）Dependencies—>(右上)绿色+号—>选择SlidingMenu_library，点击Ok,OK

6.finish();

7.API介绍：
SlidingMenu
第一种形式
直接通过 new 的形式 获取 SlidingMenu
SlidingMenu mSlidingMenu = new SlidingMenu(this);
设置 SlidingMenu 的属性 设置菜单的滑出形式 右 左 左右(存在二级菜单) setMode
mSlidingMenu.setMode(SlidingMenu.LEFT_RIGHT);
设置手势控制菜单的模式
TOUCHMODE_MARGIN 边缘
TOUCHMODE_FULLSCREEN 整个屏幕
TOUCHMODE_NONE 禁止通过手势控制菜单显示/隐藏
setTouchModeAbove
mSlidingMenu.setTouchModeAbove(SlidingMenu.TOUCHMODE_FULLSCREEN);
设置菜单的宽度 推荐使用下面俩个方法 可以通过在配置文件中设置值单位为dp 1.setBehindOffsetRes 2.setBehindWidthRes 区别 第一个方法是设置菜单距离屏幕的偏移量 （菜单宽度 = 屏幕宽度 - 偏移量） 第二个方法是直接设置菜单宽度
mSlidingMenu.setBehindWidthRes(R.dimen.slidingmenu_width);
将菜单添加到Activity指定菜单的区域 SLIDING_CONTENT 菜单在内容区域 SLIDING_WINDOW 菜单在整个屏幕 不存在ActionBar的时候 没有区别 attachToActivity
mSlidingMenu.attachToActivity(this, SlidingMenu.SLIDING_CONTENT);
设置菜单 可以指定 id 后者是 View setMenu
mSlidingMenu.setMenu(R.layout.sliding_menu_left);
